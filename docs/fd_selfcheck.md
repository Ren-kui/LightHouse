> 未填写本清单的交付 = QA 拒收（依据 `work_standards.md` W003 FD 交底 + FD 自检声明）。

---

## 使用方式

1. 复制下方表格到当次交付说明中。
2. 逐项操作并填入结果（PASS / FAIL / 不适用）。
3. 至少一条 FAIL = 修完重新过全表。
4. 将填完的表格随代码一并移交 QA。

## 交互矩阵（交叉场景）

> 任何涉及文本系统、面板、暂停、小游戏的改动，必须额外过此 6 项。
> 普通改动可跳过 X4/X6。

| # | 场景 | 操作 | 预期 |
|---|------|------|------|
| X1 | 逐字打印进行中 | 按 Tab | 打字暂停，面板滑入 |
| X2 | `[pause N]` 倒计时中 | 按 Tab | 倒计时取消，面板滑入；关面板后手动空格翻页 |
| X3 | 面板打开 + 暂停倒计时 | 等 N 秒 | 面板背后文本不翻页 |
| X4 | 小游戏运行中 | 按 Tab | 面板不弹出（或小游戏未受影响） |
| X5 | 选项显示中 | 按 Tab | 面板滑入，选项不可点继续，关面板后仍在原地 |
| X6 | 存档/读档浮窗打开 | 按 Tab | Tab 不响应（浮窗优先） |
| X7 | 小游戏中，计时器到期瞬间点击 | 光点即将消失时快速点击 | 旧光点正确消失，新光点仅出现一个，无叠层 |
| X8 | MG4A 海鸟躲避，连中 3 次直接失败 | 连中 3 次鸟 → 小游戏结束 | 回调延迟 10ms 触发，canvas 安全销毁，无 `NoneType` 崩溃 |
