## 全局技术约束

| # | 约束 | 说明 |
|---|------|------|
| 1 | **Python 标准库 only** | 不引入任何第三方库（含 pytest/pygame/numpy） |
| 2 | **Tkinter GUI** | 单线程，异步操作需 `threading` |
| 3 | **exe 最终打包** | 目标用户无需安装 Python 环境 |
| 4 | **中文注释** | 所有代码注释使用中文（D013） |
| 5 | **JSON 数据驱动** | 剧情/日记/音效配置全部 JSON，代码与内容分离 |
